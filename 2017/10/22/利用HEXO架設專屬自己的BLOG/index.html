利用HEXO架設屬於自己的BLOG

其他 blog 2017/10/22

過去曾用過WordPress、logdown等等來撰寫文章，但都一直無法讓自己滿意，不是架設太複雜就是很難去客製化自己的主題。基本主題東改改西改改，花費的時間一多，就喪失了隨手寫Blog的意義了。

在一個偶然的情況下，逛到一個技術部落格，就是使用HEXO來實現，簡潔、快速架設、美觀的優點正好是我需要的，於是這個站就這麼產生了。

常用指令：

  1. hexo generate (hexo g) 生成靜態文件，會在當前目錄下生成一個新的叫做public的文件夾。
  2. hexo server (hexo s) 啟動Local web服務，用於Hexo的預覽
  3. hexo deploy (hexo d) 部署到遠端（比如github, heroku等平台）

這邊有一個我常用的組合指令：

hexo clean && hexo d -g

它就是將上面一整串合併在一起，然後產生靜態檔，最後發布到github上。
